HTML page routing

+ [ HTML page routing ] -is: [ a technique used to navigate between different {views or pages} in a web application - without reloading the entire page ]. This is commonly implemented in Single Page Applications (SPAs) to provide a seamless user experience. There are two primary methods for routing in 'Single Page Applications': { URL routing -and: hash routing }.


hash routing

+ 'hash fragment' ej> https://sitio.com/#/"name" <-- no recarga la la página - pero puede usarse para cambiar el contenido dinámicamente
